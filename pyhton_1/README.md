# Program Descriptions

## GoFit.py

The **GoFit application** is a command-line fitness planner that asks the user for:

- Name
- Age
- Fitness goal

It then generates a **7-day workout plan**.

### Features

- Age-based workout recommendations
- Two fitness options
  - Gain weight
  - Lose weight
- Input validation
- Interactive terminal interface

Run the program using:

```bash
python3 GoFit.py
