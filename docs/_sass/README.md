# _sass

This folder contains Sass (Syntactically Awesome Style Sheets) partials used to define the styling and visual presentation of the Jekyll site. These partials are typically imported and compiled into a single CSS file by a preprocessor.

## Contents

-   **fonts.scss:** Defines font imports and styles used across the website.
-   **jekyll-theme-minimal.scss:** Core styles specific to the "Jekyll Theme Minimal," likely including overall theme structure and branding.
-   **minimal.scss:** General or foundational styles, possibly an extension or a base for the theme.
-   **rouge-github.scss:** Styles for syntax highlighting, specifically tailored for Rouge (Jekyll's default syntax highlighter) to match GitHub's aesthetic.

## Purpose

The main purpose of this folder is to organize and manage the site's stylesheets using Sass, which allows for more maintainable, modular, and feature-rich CSS development. It facilitates the creation of a consistent and appealing design for the GitHub Pages.
