# css

This folder contains the CSS (Cascading Style Sheets) files that define the visual styling and presentation of the GitHub Pages website. These stylesheets control the layout, colors, fonts, and overall aesthetic of the site.

## Contents

-   **style.scss:** The main Sass file that compiles into the primary stylesheet for the website, likely importing partials from the `_sass` directory.

## Purpose

The main purpose of this folder is to house the stylesheets responsible for the look and feel of the website, ensuring a consistent and branded user interface across all pages.
