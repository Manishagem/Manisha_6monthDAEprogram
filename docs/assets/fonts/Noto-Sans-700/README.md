# Noto-Sans-700

This folder contains the font files for the Noto Sans typeface, specifically the bold (700 weight) variant. These files are used to render text in a bold style on the GitHub Pages website, contributing to the site's overall typographic design.

## Contents

-   **Noto-Sans-700.eot:** Embedded OpenType font file for older Internet Explorer versions.
-   **Noto-Sans-700.svg:** SVG font file for older browsers.
-   **Noto-Sans-700.ttf:** TrueType Font file, widely supported.
-   **Noto-Sans-700.woff:** Web Open Font Format file, commonly used for web performance.
-   **Noto-Sans-700.woff2:** Web Open Font Format 2.0 file, offering better compression and performance.

## Purpose

The main purpose of this folder is to provide the necessary font assets for displaying bold text in the Noto Sans typeface, ensuring consistent and high-quality typography across various browsers and devices.
