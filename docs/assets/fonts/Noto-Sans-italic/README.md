# Noto-Sans-italic

This folder contains the font files for the Noto Sans typeface, specifically the italic style variant. These files are used to render text in an italic style on the GitHub Pages website, contributing to the site's overall typographic design.

## Contents

-   **Noto-Sans-italic.eot:** Embedded OpenType font file for older Internet Explorer versions.
-   **Noto-Sans-italic.svg:** SVG font file for older browsers.
-   **Noto-Sans-italic.ttf:** TrueType Font file, widely supported.
-   **Noto-Sans-italic.woff:** Web Open Font Format file, commonly used for web performance.
-   **Noto-Sans-italic.woff2:** Web Open Font Format 2.0 file, offering better compression and performance.

## Purpose

The main purpose of this folder is to provide the necessary font assets for displaying italic text in the Noto Sans typeface, ensuring consistent and high-quality typography across various browsers and devices.
