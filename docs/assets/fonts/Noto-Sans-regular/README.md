# Noto-Sans-regular

This folder contains the font files for the Noto Sans typeface, specifically the regular weight variant. These files are used to render standard text on the GitHub Pages website, forming the primary body font for the site's content.

## Contents

-   **Noto-Sans-regular.eot:** Embedded OpenType font file for older Internet Explorer versions.
-   **Noto-Sans-regular.svg:** SVG font file for older browsers.
-   **Noto-Sans-regular.ttf:** TrueType Font file, widely supported.
-   **Noto-Sans-regular.woff:** Web Open Font Format file, commonly used for web performance.
-   **Noto-Sans-regular.woff2:** Web Open Font Format 2.0 file, offering better compression and performance.

## Purpose

The main purpose of this folder is to provide the necessary font assets for displaying regular text in the Noto Sans typeface, ensuring consistent and high-quality typography across various browsers and devices.
