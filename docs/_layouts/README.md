# _layouts

This folder contains the layout templates for the Jekyll site. Layouts define the overall structure and common elements of different pages, such as headers, footers, and sidebars, into which content is rendered.

## Contents

-   **default.html:** The default layout for most pages on the site, providing a common structure.
-   **post.html:** A specific layout designed for blog posts or articles, often including elements like author information, date, and comment sections.

## Purpose

The main purpose of this folder is to centralize the design and structural components of the website, ensuring a consistent look and feel across various page types while allowing content to be dynamically inserted into these predefined structures.
