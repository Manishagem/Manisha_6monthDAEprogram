# _includes

This folder contains HTML snippets and other files that are included across the Jekyll site. These files are typically used for common elements like headers, footers, or custom scripts, allowing for reuse and consistency across the website.

## Contents

-   **head-custom-google-analytics.html:** Custom HTML for integrating Google Analytics into the site's head section.
-   **head-custom.html:** General custom HTML to be included in the `<head>` section of the site, potentially for meta tags, custom CSS, or scripts.

## Purpose

The main purpose of this folder is to store reusable components and custom elements that are included in different parts of the Jekyll-powered GitHub Pages site, ensuring modularity and easier maintenance of the web pages.
