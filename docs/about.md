# About Me

I am currently enrolled in a six-month cohort program, focusing on becoming an IAM (Identity and Access Management) Engineer. This portfolio showcases my journey and projects.

## Aspirations

I aspire to be an IAM Engineer, focusing on securing digital identities and managing access control within organizations. My goal is to develop robust and scalable identity solutions that enhance security and streamline user experiences.

## Six-Month Cohort Program

This program is designed to provide comprehensive training in various aspects of cybersecurity, with a strong emphasis on practical skills and real-world project development.

## Semester 2 Project Plan

My project plan for the coming three months involves deep diving into advanced IAM concepts, including federated identity, single sign-on (SSO), multi-factor authentication (MFA), and governance. I will be developing a practical project to demonstrate my understanding and implementation skills in these areas. More details about this project can be found in my [Projects](/projects/) section.
